## 🚀 Proyectos de Rick & Morty - JavaScript Asíncrono

¡Bienvenido! Este repositorio está diseñado para ayudarte a entender cómo funciona JavaScript, el consumo de APIs y la asincronía, comparando un desarrollo tradicional (**Vanilla**) con uno optimizado (**Antigravity**).

## 🏃 Cómo ejecutar los proyectos

Este es un proyecto de **JavaScript Puro (Vanilla JS)**, por lo que **no necesitas ejecutar `npm install`**. Los archivos funcionan directamente en el navegador.

Para ver la aplicación funcionando, recomendamos usar un "servidor local" para evitar problemas de permisos. Sigue estos pasos según la versión que quieras ver:

### 🌌 Opción A: Proyecto Antigravity (Optimizado)
Esta versión utiliza técnicas modernas y asistencia inteligente.
1. Abre una terminal (o consola) en la carpeta raíz del proyecto.
2. Escribe: `cd rick-morty-antigravity`
3. Escribe: `npx serve .`
4. Haz clic en el enlace que aparecerá (ejemplo: `http://localhost:3000`).

### 🍦 Opción B: Proyecto Vanilla (Tradicional)
Esta versión es JavaScript puro, ideal para entender las bases.
1. Abre una terminal en la carpeta raíz.
2. Escribe: `cd rick-morty-vanilla`
3. Escribe: `npx serve .`
4. Haz clic en el enlace que aparecerá.

## 🖥️ El uso de la Consola (F12)

JavaScript es un lenguaje que ocurre "detrás de cámaras". Para ver lo que el código está haciendo, usamos la **Consola del Navegador**.

- **¿Cómo abrirla?**: Presiona la tecla **F12** en tu teclado (o haz clic derecho en la página -> **"Inspeccionar"** -> pestaña **"Console"**).
- **¿Qué buscar allí?**: 
  - Verás **listas de personajes** que vienen de la API. Puedes hacer clic en las flechitas para ver sus detalles (nombre, especie, etc.).
  - Podrás ver si hay algún error (mensajes en rojo) que necesites corregir.
